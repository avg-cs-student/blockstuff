# Blocks

Blocks is a small library of components for building games.

__Examples__:
```
┌─┐ ┌─┬─┐ ┌─┬─┐ ┌─┬─┬─┐ ┌─┬─┬─┐
└─┘ ├─┼─┤ ├─┼─┤ ├─┼─┼─┤ ├─┼─┼─┤
    └─┴─┘ ├─┼─┤ └─┴─┴─┘ ├─┼─┼─┤
          └─┴─┘         └─┴─┴─┘
  ┌─┐  
┌─┼─┼─┐
└─┴─┴─┘

┌─┬─┐    ┌─┬─┬─┐   ┌─┬─┬─┬─┐  ┌─┬─┬─┬─┬─┐
└─┴─┘    └─┴─┴─┘   └─┴─┴─┴─┘  └─┴─┴─┴─┴─┘
```

See my [TUI](https://github.com/avg-cs-student/jcblocktui) for example usage.
